# Changelog

## [0.1.0] - 2026-03-24
### Added
- Initial release
- CLI commands: init, save, goto, list, branches, diff, gc
- Go SDK for programmatic use
- Content-addressable object store with deduplication
- Timeline DAG engine with automatic branching
- Atomic writes and crash recovery

### Known Limitations
- No cloud sync (local only)
- No GUI (CLI + Go SDK only)
- No execution replay (Phase 2)
